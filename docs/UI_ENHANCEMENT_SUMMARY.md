## 🎨 统一设计元素

### 1. 色彩系统

**主色调**：
```css
--primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
--success-gradient: linear-gradient(135deg, #10b981 0%, #34d399 100%)
--warning-gradient: linear-gradient(135deg, #f59e0b 0%, #fbbf24 100%)
--danger-gradient: linear-gradient(135deg, #ef4444 0%, #f87171 100%)
--info-gradient: linear-gradient(135deg, #3b82f6 0%, #60a5fa 100%)
```

**背景色**：
```css
--bg-gradient: linear-gradient(135deg, #f8fafc 0%, #f1f5f9 100%)
```

### 2. 圆角系统
- 卡片：12px
- 按钮：6-8px
- 输入框：6px
- Tag：4px

### 3. 阴影系统
- 默认：`0 2px 12px 0 rgba(0, 0, 0, 0.08)`
- 悬停：`0 4px 20px 0 rgba(0, 0, 0, 0.12)`
- 按钮悬停：`0 2px 8px rgba(0, 0, 0, 0.15)`

### 4. 动画效果
- 页面淡入：fadeIn 0.3s
- 卡片悬停：上浮 2px
- 按钮悬停：上浮 1px + 阴影
- 表格行悬停：浅蓝背景

### 5. 字体优化
- 标题：600-700 font-weight
- 正文：400-500 font-weight  
- 数字：Segoe UI 字体
- 中文优先：PingFang SC

## 💡 使用建议

### 自定义主题色

如需修改主题色，编辑 `frontend/src/main.ts`：

```typescript
:root {
  --primary-gradient: linear-gradient(135deg, #YOUR_COLOR1 0%, #YOUR_COLOR2 100%);
}
```

### 调整动画速度

如需加快/减慢动画，修改页面CSS：

```css
transition: all 0.3s;  /* 改为 0.2s 或 0.4s */
```

### 禁用动画

如需禁用所有动画效果，在全局样式添加：

```css
* {
  animation: none !important;
  transition: none !important;
}
```
